ZimTrust API

Integrate milestone escrow into your own app or workflow. RESTful JSON API with webhooks.

Reference
Authentication Projects Milestones Evidence Webhooks

Authentication

All API requests require a Bearer token in the Authorization header. Generate API keys from your ZimTrust dashboard under Settings → API.

Authorization: Bearer zt_live_xxxxxxxxxxxxxxxxxxxx

Projects

GET/v1/projects
List all projects for the authenticated user.
POST/v1/projects
Create a new project with milestones.

Milestones

PATCH/v1/milestones/:id/approve
Approve a milestone and trigger payout. Milestone must be in NEEDS_REVIEW status.

Webhooks

Configure webhook endpoints in your dashboard. ZimTrust posts events to your URL on every status change.

Event types
milestone.needs_review  ·  milestone.approved  ·  milestone.paid  ·  payout.completed  ·  dispute.raised
